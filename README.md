## React Native Seed App

### Build the project
```
npm install
npm start

react-native run-ios (separate command line)
react-native run-android (separate command line)
```

### Run unit tests
```
npm test
```

### Run E2E tests
```
(IOS)
npm run build:e2e:ios
npm run test:e2e:ios

---------------------------

(ANDROID)
npm run build:e2e:android
npm run test:e2e:android
```
